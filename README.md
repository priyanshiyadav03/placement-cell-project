# Placement Prediction System

## Live Demo(Deployment Link)
https://placement-cell-project-gqbc9npmte2hbz3tdh5vzo.streamlit.app/



---

A Machine Learning based Placement Prediction Web App built using Streamlit and Scikit-learn.

## Features

- Predicts whether a student is likely to be placed
- Interactive Streamlit UI
- Uses trained ML model (`placement_model.pkl`)
- Supports multiple student attributes

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- NumPy
- Pandas
